Token.io MCP API

This is a MCP (Model Context Provider) API server for Token.io banking services.

Available Endpoints

Health Check

GET /health

Check if the API server is running properly.

List Available Tools

GET /tools

Returns a list of available tools and their parameters.

Get Banks

POST /tools/tokenio-get-banks

Retrieves a list of banks with optional filtering.

{
  "search": "barclays",
  "countries": ["GB"]
}

Get Bank Details

POST /tools/tokenio-get-bank-details

Gets detailed information about a specific bank.

{
  "bankId": "ob-barclays"
}

Integration

This API can be integrated with AI assistants like Perplexity or GPT to provide banking data capabilities.
